# 📦 Changelog

Todas as mudanças relevantes do **Charleston Design System** serão documentadas neste arquivo.

Seguimos o padrão **Keep a Changelog** e **SemVer**.

## [0.1.0] - 2026-01-17

### 🎉 Added

- Estrutura inicial do Design System
- ThemeProvider com contrato de tema
- Sistema de tokens (cores, spacing, radius, tipografia)
- Componentes base:
    - Button
    - Text
    - Stack
    - Screen
    - Toolbar
    - Card
    - Input
    - Checkbox / Radio / Switch
    - Badge
    - Image
    - Skeleton
    - Dialog
    - BottomSheet
    - ActionSheet
    - Toast
    - EmptyState / ErrorState
- Build com tsup
- Publicação NPM privada
