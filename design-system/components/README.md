# Components (`src/components/`)

Diretório principal da biblioteca de componentes de interface do **Charleston Design System**.

## 📂 Organização

Cada componente reside em sua própria pasta e segue um padrão rigoroso de arquivos (ex: `index.tsx`, `*.root.tsx`, `*.styles.ts`).

### Lista de Componentes

- **`accordion`**: Exibição de conteúdo expansível em painéis.
- **`action-sheet`**: Menu deslizante para ações rápidas.
- **`alert`**: Diálogos de alerta e feedback.
- **`aspect-ratio`**: Componente para manter proporções de imagem/vídeo.
- **`avatar`**: Representação visual de usuários.
- **`badge`**: Indicadores de status ou contagem.
- **`bottom-navigation`**: Barra de navegação inferior.
- **`bottom-sheet`**: Painéis modais que deslizam da base.
- **`button`**: Componentes de botão variados.
- **`card`**: Contêineres de conteúdo estruturado.
- **`carousel`**: Exibição de itens em carrossel.
- **`checkbox`**: Seleção múltipla.
- **`collapsible`**: Elementos de interface que podem ser ocultos/exibidos.
- **`dialog`**: Janelas modais de diálogo.
- **`divider`**: Separadores visuais.
- **`empty-state`**: Estados para listas vazias.
- **`error-state`**: Feedback visual de erros.
- **`floating-button`**: Botão flutuante para ações de destaque.
- **`image`**: Componente de imagem otimizado.
- **`input`**: Campos de entrada de texto.
- **`list`**: Componentes de lista e itens de lista (`tile`).
- **`loading-overlay`**: Indicador de carregamento sobreposto.
- **`otp-input`**: Campo de entrada para códigos de verificação (One-Time Password).
- **`permission-gate`**: Controle de acesso por permissões.
- **`portal`**: Renderização de componentes fora da hierarquia DOM principal.
- **`pressable`**: Base para elementos clicáveis.
- **`progress-bar`**: Indicadores de progresso linear.
- **`radio`**: Seleção única.
- **`screen`**: Contêiner base para telas.
- **`select`**: Menu de seleção de opções.
- **`skeleton`**: Placeholders de carregamento.
- **`slider`**: Seleção de valores em uma faixa contínua.
- **`stack`**: Layouts lineares (HStack, VStack).
- **`step-indicator`**: Indicador visual de progresso em etapas.
- **`stepper`**: Controle numérico incremental.
- **`switch`**: Alternadores de estado ligado/desligado.
- **`tabs`**: Organização de conteúdo em abas.
- **`text`**: Tipografia básica do sistema.
- **`toast`**: Notificações temporárias.
- **`toolbar`**: Barras de ferramentas superiores/inferiores.
- **`tooltip`**: Pequenas mensagens informativas ao pairar ou focar.
