# Theme (`src/theme/`)

Este diretório gerencia a aparência visual do **Charleston Design System**, incluindo temas Light e Dark.

## 📂 Conteúdo

- **`index.ts`**: Ponto de entrada para o provedor de tema e hooks relacionados.
- **Configurações de Tema**: Lógica para alternância e persistência do tema (Light/Dark).
- **Integração com Tokens**: Mapeamento dos design tokens para variáveis semânticas de tema.

## 📜 Diretrizes

Consulte [CODE_RULES.md](./CODE_RULES.md) para saber como estender o tema ou adicionar novas variáveis semânticas.

> [!IMPORTANT]
> **Manutenção da Documentação:** Este README deve ser atualizado sempre que houver mudanças na lógica de temas ou na estrutura de arquivos desta pasta.
