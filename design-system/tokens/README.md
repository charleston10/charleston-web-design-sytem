# Design Tokens (`src/tokens/`)

Design tokens são a "única fonte de verdade" para valores visuais como cores, tipografia e espaçamentos no **Charleston Design System**.

## 📂 Tokens Disponíveis

- **`colors.ts`**: Definição da paleta de cores.
- **`radius.ts`**: Valores de arredondamento de bordas.
- **`spacing.ts`**: Escala de espaçamentos internos e externos.
- **`typography.ts`**: Definições de tamanhos de fonte, pesos e alturas de linha.
- **`index.ts`**: Exportação centralizada de todos os tokens.

## 📜 Diretrizes

Tokens literais (ex: `blue-500`) devem ser definidos aqui e depois mapeados para nomes semânticos no `src/theme/`.
